## 주요 기능

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 홈 | `/` | 인기 여행지, 국가 탐색, 테마별 추천 |
| 여행지 탐색 | `/destinations` | 국가/테마 필터, 즐겨찾기, 일정 추가, **환전소 바로가기** |
| 여행지 상세 | `/destination/:id` | 명소, 팁, 교통, 음식 정보, **여행 준비 도구** |
| 일정 플래너 | `/planner` | 일별 일정 추가/편집/저장, **빠른 도구 (환전소/회화)** |
| 여행 팁 | `/tips` | **실시간 환율**, **환전소 찾기**, 회화, 비자, 짐 챙기기, 긴급 정보 |
| 마이 여행 | `/my` | 저장 일정, 즐겨찾기, 최근 조회, 데이터 관리 |

### 💴 환전 & 환율 기능
- **실시간 환율**: JPY, THB, VND, TWD, SGD, IDR, MYR, HKD 8개 통화
- **환전 계산기**: 원화 기준 각 통화별 환산
- **환전소 정보**: 20+ 도시별 추천 환전소 (주소, 영업시간, 환율 우대 팁)
- **지원 도시**: 도쿄, 오사카, 교토, 삿포로, 방콕, 푸켓, 치앙마이, 파타야, 다낭, 하노이, 호치민, 나트랑, 타이베이, 타이중, 가오슝, 싱가포르, 발리, 자카르타, 쿠알라룸푸르, 페낭, 홍콩

## 데이터 저장 기능
- **즐겨찾기**: 하트 버튼으로 여행지 저장
- **여행 일정**: 일별 스케줄 생성/편집 (localStorage)
- **내보내기**: JSON 백업, CSV (엑셀), 텍스트 일정표
- **가져오기**: JSON 파일 복원
- **PWA 설치**: 홈 화면 추가, 오프라인 지원

## 기술 스택
- **Backend**: Hono (TypeScript)
- **Frontend**: Tailwind CSS + Material Icons (CDN)
- **스토리지**: localStorage (즐겨찾기, 일정)
- **배포**: Cloudflare Pages
- **빌드**: Vite + @hono/vite-cloudflare-pages
